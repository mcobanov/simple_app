# simple_app

## Project setup

###  Navigate to source folder and install dependencies by running:
```
npm install
```

---
### Start dev server:
```
npm run dev
```
* App will be available at http://localhost:5200/

---
### Compiling for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
